# JPA Tutorials Repository

This repository contains tutorials and practice code for Java Persistence API (JPA) using Spring Boot and Hibernate. It includes examples of entities, repositories, and complex queries.

## Features
- Spring Data JPA implementations
- Various entity mappings (OneToOne, OneToMany, ManyToMany)
- Custom queries and JPQL
- Interview prep notes

## Technologies
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven
